Root-Finding Algorithms for Polynomial Equations

numerical analysis root finding Newton's method Bisection method Secant method HTML MATLAB/Objective-C
H
Hila Shmuel <> to Hila Shmuel <>
Numerical Analysis HW5
Mon, 18 Dec 2017 12:10:35 +0200
The provided code defines functions for each method (Newton, Secant, and Bisection) to calculate the root of a given function. The user can specify the range, initial guesses, tolerance levels, and maximum iterations for the calculations. The methods take a symbolic function and utilize numerical techniques to converge on a solution, returning information such as the root value, number of iterations, and error estimates. The main script (HW5.m) executes these methods on the specified polynomial function and provides tests to validate the algorithms' performance.
