Audio SNR Analysis

audio processing MATLAB signal-to-noise ratio waveform analysis MATLAB/Objective-C
H
Hila Shmuel <>
Image and Audio Compression
Wed, 12 May 2010 18:57:41 +0300
The code consists of two main MATLAB scripts and a function for calculating the signal-to-noise ratio. The first script reads an original audio file and a low-pass filtered version, calculates their SNR, and displays the result. The second script reads an original audio file and three different versions that have been subjected to varying levels of Gaussian noise, computes their SNR values relative to the original file, and outputs these values. The SNR is computed using a custom function that determines the power of the signal and noise.
