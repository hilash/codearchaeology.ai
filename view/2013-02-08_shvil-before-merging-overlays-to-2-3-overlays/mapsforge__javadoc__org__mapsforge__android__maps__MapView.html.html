<DT><PRE>public class <B>MapView</B><DT>extends android.view.ViewGroup</DL>
</PRE>

<P>
A MapView shows a map on the display of the device. It handles all user input and touch gestures to move and zoom the
 map. This MapView also includes a scale bar and zoom controls. The <A HREF="../../../../org/mapsforge/android/maps/MapView.html#getController()"><CODE>getController()</CODE></A> method returns a
 <A HREF="../../../../org/mapsforge/android/maps/MapController.html" title="class in org.mapsforge.android.maps"><CODE>MapController</CODE></A> to programmatically modify the position and zoom level of the map.
 <p>
 This implementation supports offline map rendering as well as downloading map images (tiles) over an Internet
 connection. The operation mode of a MapView can be set in the constructor and changed at runtime with the
 <A HREF="../../../../org/mapsforge/android/maps/MapView.html#setMapGeneratorInternal(org.mapsforge.android.maps.mapgenerator.MapGenerator)"><CODE>setMapGeneratorInternal(MapGenerator)</CODE></A> method. Some MapView parameters depend on the selected operation mode.
 <p>
 In offline rendering mode a special database file is required which contains the map data. Map files can be stored in
 any folder. The current map file is set by calling <A HREF="../../../../org/mapsforge/android/maps/MapView.html#setMapFile(java.io.File)"><CODE>setMapFile(File)</CODE></A>. To retrieve the current
 <CODE>MapDatabase</CODE>, use the <A HREF="../../../../org/mapsforge/android/maps/MapView.html#getMapDatabase()"><CODE>getMapDatabase()</CODE></A> method.
 <p>
 <A HREF="../../../../org/mapsforge/android/maps/overlay/Overlay.html" title="class in org.mapsforge.android.maps.overlay"><CODE>Overlays</CODE></A> can be used to display geographical data such as points and ways. To draw an overlay on top
 of the map, add it to the list returned by <A HREF="../../../../org/mapsforge/android/maps/MapView.html#getOverlays()"><CODE>getOverlays()</CODE></A>.
